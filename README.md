# CLI-Rwanda

Website for **Consilium for Local Initiatives (CLI-Rwanda)** — a Rwanda-based consulting firm that strengthens the sustainability, impact, and growth of local initiatives across the nonprofit and social enterprise ecosystem.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- React 19
- Tailwind CSS 4
- Framer Motion
- next-themes

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start the dev server     |
| `npm run build` | Production build         |
| `npm run start` | Serve the production app |
| `npm run lint`  | Run ESLint               |

## Pages

- `/` — Home
- `/about` — About the firm
- `/services` — Consulting services
- `/contact` — Contact form and details
